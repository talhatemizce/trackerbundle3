📚 BookDepot → TrackerBundle3

bookdepot.com'daki kitapları tek tıkla TrackerBundle3 envanterine aktar

✅ Backend endpoint'leri hazır!
POST /bookdepot/import ve GET /bookdepot/inventory endpoint'leri app/main.py'e eklendi.
Sadece VPS URL'ini gir ve bookmarklet'i bookmark çubuğuna sürükle.
1 VPS API URL'ini Gir

VPS'in HTTP URL'si (Cloudflare tunnel varsa HTTPS URL'si):

bookdepot.com HTTPS kullandığı için HTTP URL ile çalışmaz. Cloudflare Tunnel gerekiyorsa 2. adıma bak.

2 Bookmarklet'i Sürükle
📦 BookDepot → TB3

👆 Bu butonu tarayıcının yer imi/bookmark çubuğuna sürükle
(Yer imi çubuğu yoksa: Chrome → Ctrl+Shift+B)

← Önce URL gir ve "Bookmarklet Oluştur"a tıkla
3 Cloudflare Tunnel (Sadece HTTPS Gerekiyorsa)

bookdepot.com HTTPS → VPS HTTP bağlantısı tarayıcı tarafından bloklanır.
Eğer VPS'te zaten HTTPS/nginx yoksa bu komutu çalıştır:

curl -L -o ~/cloudflared https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 chmod +x ~/cloudflared nohup ~/cloudflared tunnel --url http://localhost:8000 > /tmp/cf.log 2>&1 & sleep 5 && grep -o 'https://[^ ]*\.trycloudflare\.com' /tmp/cf.log | head -1

Son satırda çıkan https://xxx.trycloudflare.com URL'ini 1. adıma yapıştır.

4 Nasıl Kullanılır
  1. bookdepot.com'da herhangi bir arama veya kategori sayfasına git
  2. Yer imi çubuğundaki 📦 BookDepot → TB3'e tıkla
  3. Sayfadaki tüm kitaplar otomatik taranır ve VPS'e gönderilir
  4. Sağ üstte yeşil ✅ bildirim kaç kitap eklendiğini gösterir
  5. Birden fazla sayfa varsa her sayfada tekrar tıkla
